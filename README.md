# ⚡ SportLife — Сайт про спортивный образ жизни

Современный одностраничный сайт о спорте, тренировках, питании и мотивации.

## 🚀 Функции

- 🏋️ Каталог тренировок (для начинающих, продвинутых)
- 🥗 Раздел правильного питания с советами
- 🔥 Ежедневная мотивация и цитаты
- 📬 Форма обратной связи
- 📱 Полностью адаптивный дизайн

## 🛠 Технологии

- HTML5
- CSS3 (Flexbox, Grid, CSS-анимации)
- Vanilla JavaScript
- Google Fonts (Montserrat)

## 📂 Структура

```
sports-lifestyle/
├── index.html
├── style.css
├── script.js
└── README.md
```

## 🌐 Запуск

Открой `index.html` в браузере — и готово!
